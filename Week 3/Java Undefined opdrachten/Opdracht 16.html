Opdracht 16:

1. charAt().
const sentence = 'The quick brown fox jumps over the lazy dog.';
const index = 4;
console.log(sentence.charAt(index)); Returns het 4de karakter van de string. in dit geval "q".
2. concat().
const str1 = 'Hello';
const str2 = 'World';
console.log(str1.concat(' ', str2)); geeft: "Hello World"
console.log(str2.concat(', ', str1)); geeft: "World, Hello"
concat voegt de strings samen en maakt er een nieuwe string van.
3. slice().
let str1 = 'The morning is upon us.';
str2 = str1.slice(1, 8);
str3 = str1.slice(4, -2);
console.log(str2); Antwoord is: he morn.
console.log(str3); Antwoord is: morning is upon u
Bij Slice kunnen delen van een string geselecteerd worden. De nummers geven aan wat geselecteerd wordt.
4. split().
const str = 'The quick brown fox jumps over the lazy dog.';
const words = str.split(' ');
console.log(words[3]); Resulteerd in "fox"
Met split wordt het vierde woord geretourneerd. Begin tellen met "0".
5. substr().
const str = 'Mozilla';
console.log(str.substr(1, 2)); Retourneerd: "oz"
console.log(str.substr(2)); Retourneerd: "zilla"
Retourneerd een deel van de tekenreeks, beginnend bij de opgegeven index en daarna een bepaald aantal tekens.
