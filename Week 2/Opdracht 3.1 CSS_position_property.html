Opdracht 3.1 Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

* Static - Standaard. Een element met position: static heeft altijd de positie die de normale weergave van de pagina het geeft. (Een static element negeert alle top, bottom, left, of right declarations)
* Relative - Een element met position: relative verplaatst een element relatief ten opzichte van de normale positie, dus "left:20" voegt 20 pixels toe aan de linker positie van het element
* Absolute - Een element met position: absolute is gepositioneerd op de gespecificeerde coordinaten relatief naar het block waarin het zich bevindt. De positie van het element is gespecificeerd met de "left", "top", "right", and "bottom" properties
* Fixed - Een element met position: fixed is gepositioneerd op de gespecificeerde coordinaten relatief naar het browser venster. De positie van het element is gespecificeerd met de "left", "top", "right", en "bottom" properties. Het element blijft op die positie onafhankelijk van scrolling.
* Sticky - Een sticky element beweegt zich tussen een relative en een fixed position, afhankelijk van de scroll position. Het is relative tot een bepaalde positie wordt bereikt in het venster. Dan wordt het fixed.

Standaard staat de default positie op Static.
