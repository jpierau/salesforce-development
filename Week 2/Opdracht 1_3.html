Opdracht 1.3

Er zijn verschillende soorten type Selectors, maak voorbeelden van type selectors, class selectors en descendant selectors

De meest bekende selector (voor mij dan) is de type selector. Hier een voorbeeld: <style> h3 { text-align: center; color: red; } </style>
Voordeel, via deze selector krijgen alle h3 tags automatisch gecentreerd met rode tekst.

De class selector ziet er als volgend uit:

<style>
.center { text-align: center; color: blue; }
</style>
Voordeel is, via deze selector is automatisch alle tekst blauw en gecentreerd.

De descendant selector als volgt uit

<div>
<h2>Title</h2>
<p>paragraph tekst</p>
tekst zonder tag
</div>
Voordeel hier aan is is dat je meerdere elementen in 1 keer kan aanpassen.
