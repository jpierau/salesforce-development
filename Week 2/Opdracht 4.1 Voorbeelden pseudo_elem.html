Opdracht 4.1 Geef 4 vb van Pseudo elementen met beschrijving

  • ::after
  • ::first-letter
  • ::first-line
  • ::placeholder

1. Het after element laat je content plaatsen achter het gemaakte element
2. Het first-letter element breng je style aan bij het eerste letter.
3. Het first-line element dan style eerste zin van het element.
4. Het placeholder element om bijvoorbeeld de tekst in een formulier te stylen.
